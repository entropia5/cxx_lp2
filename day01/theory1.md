Оператор - единица кода; 
Используются для:
1. Вызов функций
2. Объявление и инициализация переменных
3. Работа с выражениями 

Функция - более крупная единица кода. 
Может содержать в себе множество операторов.
1. Предназначена для повторного использования или вызова, например другим оператором.
2. main является основной точкой входа в любую программу на С++ и мы не вызываем её сами.
main вызывается ОС при первом запуске моей программы. 

Переменная содержит значение или значения для последующего использования. 
Объявление перед использованием.
Затем переменная может быть использована для указания ее значения в операторах и выражениях далее в коде.

Идентификаторы.
Предоставляют удобочитаемые имена для переменных, ф-ций, классов, объектов,меток и определённых типов. 
в С++ индентификаторы состоят из букв и цифр в наборе ограничений:
1. 26 букв литинскго алфавита ISO как в строчном так и в верхнем регистре.
2. 10 арабских цифр и символ подчеркивания ASCII.
Идентификатор не может начинаться с цифры.

Variables//Переменные.
Переменные строго типизировны в С++...А в языке С это значит, что токен, представляющий переменную - представляет как её значение, так и её тип.
Когда компилятор читает код, он разбивает его на токены — минимальные смысловые куски.
Когда компилятор видит имя переменной, например х, то он знает:
1. её тип
2. где в памяти лежит её значение

Имя переменной — это не просто слово. Это некий «ярлык», за которым стоят:
1. тип 
2. место в памяти 
3. текущее значение

int i {}; //как пример, что в файле variables.cpp, в ф-ции main.
Это оператор, и этот оператор является определением переменной.
Он определяет имя и тип, а также выделяет пространство такого размера чтобы вместить значение этого типа.
и здесь ещё есть скобки. В данном случае фигурные...
Пустые инициализируют переменную придавая ей начальное значение = 0.
Без инициализации, то есть без {}, значение будет неопределенным, что может образовать ошибки.
Поэтому хорошей практикой является инициализация после определения. Пустые {} это весьма простой и хороший способ сделать это.

в традиционном способе можно было сделать это так:
int i = a;

{} это современный способ инициализации переменных.
пустые скобки обнуляют любой тип. Они для нулевой инициализации.
После инициализации мы можем изменить значение. 
например, если мы ниже int i {}; укажем i = 47; то знак = это оператор присваивания который является и знаком равенства и присваивает значение переменной. 

В С++ также есть квалификаторы.
Самый известный это const.
const сообщает компилятору что значение переменной не может изменяться после ее определения.
const int i {} - здесь int уже не просто целое число, оно уже постоянное целое число или "постоянная переменная".
Квалификатор становится частью типа.

auto.
Автотип. Позволяем компилятору определить тип по инициализации.
