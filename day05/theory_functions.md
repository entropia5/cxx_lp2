Функция
Это про управляемый блок поведения, который можно вызвать по имени.
Она может:
- что-то вернуть
- изменить аргументы
- изменить глобальное состояние
- просто что-то сделать (логирование, вывод, изменение объекта)


void-функция
- получает управление через стек вызова
- получает параметры
- создаёт локальную область видимости
- возвращает управление вызывающему коду
Просто она не возвращает значение, а возвращает контроль выполнения.


int& value можно менять
const int& value нельзя 




int& value — это не копия.

Ссылка — это другое имя для той же переменной.

Никакой новой памяти не создаётся.
value и a — это фактически один и тот же объект.



Передача по значению:
main:    a = 5
Inc1:    value = 5  (копия)




Передача по ссылке:
main:    a = 5
Inc2:    value ───► тот же самый a
Никакой копии. Один объект.
