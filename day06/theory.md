Классы рассматриваются как пользовательские типы данных.
Как любой тип, предоставляемый языком С++

Можем использовать класс для создания объекта. 
Иногда объек называют экземпляром класса.


конструкторы и деструкторы это спец члены ф-ции, которые вызываются при создании или уничтожении объекта.

Классы основаны на структурах С.

В C++ есть встроенные типы:

int

double

char

bool


но мы можем использовать свои типы:

class Player
{
public:
    int health;
    int ammo;
};
Player — такой же тип, как int.



Мы можем создать переменную этого типа:
Player — такой же тип, как int.


Это называется:

объект или экземпляр класса;

Объект — это данные + поведение;
Класс обычно содержит:
1. Данные: int health;
Это поля или члены данных.
2. Функции:
void Shoot()
{
    ammo--;
}
Это методы или функции-члены.

класс объединяет данные + функции


Конструктор
Конструктор — это функция, которая инициализирует объект.


class Player
{
public:

    int health;

    Player()
    {
        health = 100;
    }
};

теперь: Player p;

Схематично: 
1 создаётся объект
2 вызывается конструктор
3 health = 100


Деструктор
вызывается когда объект уходит из области видимости.

Пример:
class Player
{
public:

    ~Player()
    {
        std::cout << "Player destroyed\n";
    }
};

Когда объект уничтожается:
вызывается деструктор





Классы и структуры

В C++:
struct
class

struct → public
class  → private

Пример:
struct A
{
    int x; // public
};

class B
{
    int x; // private
};



Инкапсуляция
Мы скрываем данные внутри класса.

Пример,

class Player
{
private:
    int health;

public:
    void TakeDamage(int damage)
    {
        health -= damage;
    }
};

Теперь нельзя написать:
player.health = -1000





Память объекта
Когда создаётся объект: Player p;

в памяти выделяется место для всех его полей.

Например:

Player
{
int health
int ammo
}

Объект будет занимать:

sizeof(int) + sizeof(int)

Функции не лежат внутри объекта.

Они существуют отдельно.

Как это выглядит в памяти:
Stack:

p
 ├─ health
 └─ ammo

Методы класса:

Player::Shoot()
Player::Reload()

существуют в коде программы.


Без классов программа превращается в хаос.

состояние
поведение
