Переменная - единица в С++ которой можно манипулировать и хранить данные.


// #include <iostream> /директива препроцессора для подключения библиотеки ввода-вывода

// iostrem - это стандартная библиотека C++, которая предоставляет функции для ввода и вывода данных. Она включает в себя классы и функции для работы с потоками ввода-вывода, такими как std::cin для ввода с клавиатуры и std::cout для вывода на экран.

 

#include <iostream>

int main()
{
    int a;
    int a = 5;                   // объявление и инициализация переменной a. В a теперь хранится значение 5.
    int b = 10;                  // объявление и инициализация переменной b. В b теперь хранится значение 10.
    std::cout << a << std::endl; // вывод значения переменной a на экран
    std::cout << b << std::endl; // вывод значения переменной b на экран

    return 0;
}







Преобразование типов на С++ — это изменение типа данных.
int a = 5;
double b = a;

или

int a = 5;
double b = a + 0.5;




int → double
5   → 5.0

Компилятор сам сделал преобразование. Это называется:
Неявное преобразование (implicit conversion)

Минусы неявных преобразований.
Потеря данных.
double x = 5.7;
int y = x;


y=5
то есть .7 потеряется
это назыв narrowing conversion


Явное преобразование (explicit)
Когда программист сам указывает преобразование.

Лучший способ:
static_cast

double x = 5.7;
int y = static_cast<int>(x);


есть ещё
const_cast

убирает const

reinterpret_cast

очень низкоуровневое преобразование
(используется в движках)

dynamic_cast
для полиморфизма



Если в выражении участвуют разные типы, компилятор делает неявное преобразование типов (implicit conversion).

Компилятор выполняет выражение по типам операндов, а не по типу переменной, куда кладём результат.
Тип результата выражения определяется операндами, а не переменной слева.

Если хотя бы один операнд — double, всё выражение становится double.

В C++ для проверки чётности используют остаток от деления %
Например: 
10 % 2 = 0
7 % 2 = 1

То есть:
если остаток 0 → число чётное
если остаток 1 → нечётное
