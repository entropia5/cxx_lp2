vector 
контейнер с произвольным доступом, который можно использовать во многих месах, где можно использовать массив С, но с гораздо большей функциональностью.

list
список с двумя связями, оптимизированный для быстрой вставки и удаления.

set
контейнер упорядоченной последовательности в котором элементы хранятся в строгом порядке на основе их значения;

map
ассоциативный контейнер В котором уникальные ключи используются для доступа к связанным значениям

Stacks and queues 
Является двухсторонними очередями, являются очередями last in first out, first in first out, а также двухсторонними очередями для отправки и извлечение данных из стека

string
Специализированный контейнер с символов. Оптимизированный для использования в качестве строк ioastream для чтения и записи данных в файлы и устройства ввода вывода из них

cout
