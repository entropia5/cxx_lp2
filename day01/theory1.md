Оператор - единица кода; 
Используются для:
1. Вызов функций
2. Объявление и инициализация переменных
3. Работа с выражениями 

Функция - более крупная единица кода. 
Может содержать в себе множество операторов.
1. Предназначена для повторного использования или вызова, например другим оператором.
2. main является основной точкой входа в любую программу на С++ и мы не вызываем её сами.
main вызывается ОС при первом запуске моей программы. 

Переменная содержит значение или значения для последующего использования. 
Объявление перед использованием.
Затем переменная может быть использована для указания ее значения в операторах и выражениях далее в коде.

Большинство файлов С++ имеют одну или несколько директив include.
Это способ импорта библиотек для использования в коде. 
Библиотека - отдельный файл. Некоторые с расширениями, например .h 
Системные библиотеки обычно без расширений. 
Директива #include включает весь файл в процесс компиляции.

ПРОГРАММА С++.
Программа представляет собой набор операторов и выражений обычно организованных в функции и классы.
// однострочные комментарии, игнорируются препроцессором 
/**/ многострочные комментарии
Ф-ция main может иметь аргументы для разбора командной строки: int main (int argc, char** argv)
Первый аргумент (int argc) это целое число представляющее кол-ство аргументов командной строки. 
2. char** argv - массив строк, иногда так записывается char* argv[]

Тело ф-ции заключено в {}, код между скобками выполняется при каждом вызову ф-ции.
Ф-ция main возвращает целочисленное значение, 0 указывает об успешном завершении. 

Директива #include предписывает препроцессору включить ещё один заголовочный файл для передачи компилятору
в программе hello мы подключаем iostream. Файл этот поддерживает объект cout и другие классы и ф-ции связанные с вводом и выводом.
обычно это был сначала printf и потом std::cout для вывода строки в поток. 

В С++20 появился форматер std::format
/* 
const char* str {"Hello! "};
std::cout << std:format ("The str is {}\n", str);
*/
Находится в заголовочном файле #include <format>
Будем использовать для форматирования строк.

В С++23 будет функция print которая использует тот же форматировщик в качестве format
/*
const char* str {"Hello! "};
std::print("The str is {}\n", str);
*/

std - standard namespace — пространство имён стандартной библиотеки C++.
:: - двойное двоеточие является оператором обл видимости, сообщает компилятору использовать это пространство имён при поиске этого символа.
Этот оператор использует объект cout из std namespace
оператор using std::cout; - импорт имени из namespace чтобы его использовать в коле без пространства имён и оператора scope 

Что такое statement (оператор, инструкция)
В C++ statement — это инструкция, которую выполняет программа.
Она аналогична строке кода, но технически это единица выполнения.
Всегда заканчивается точкой с запятой ;
