Типы данных.

C++ - строго типизирован. Это значит, что каждое значение имеет определенный тип. 
Тип данных определяет размер. Также определяет интерпретацию значения.
К примеру, int означает присвоение целочисленного значения. А тип unsigned int - целое число того же размера без бита, зарезервированного для знака.

Базовые типы:
1. integer types
2. Floating-point types
3. Boolean types
4. Unions (Составные типы служат контейнерами для примитивов/базовых типов и других типив.)

Массив/ Array

int x[5] {1, 2, 3, 4, 5}; 
массив - это непрерывное последовательное множество объектов одного типа.
Массивы также являются основой С-строк и классов-контейнеров С++ STL.

char s[] { "this is a sting"}; 
Массив символов выполняет двойную ф-цию, поскольку примитивная строка, часто называаемая С-строкой, рассматривается как строка во многих контекстах. 

Структура/ struct.

    struct x {
    int a;
    float b;
    char c[25];
    };

Структура - последовательный набор объектов различных типов. Структура может содержать скаляры, массивы, и даже другие структуры и классы.

Классы/ class.

    class x {
    int a;
    float b;
    char c[25];
public:
    int getvalue() const;
    void setvalue (int v);
    };
Классы С++ основаны на структурах С. 
Технически класс - это структура; Частный член.
На практике класс - это структура, содержащая как члены ф-ции, так и члены данных. 
Класс может быть записан в виде структуры:

    struct x {
    int getvalue () const;
    void setvalue (int v);
private:
    int a;
    float b;
    char c[25];
    };


Объединение / union.

    union x {
    int a;
    float b;
    char c[25];
};

union - это контейнер перекрывающихся объектов;
Это позволяет одному контейнеру одновременно хранить объекты разных типов, повторно используя одно и то же пространство. 


Перечисление / enum.

    enum Color {red, green, blue};
    Color x = green;

enum - это тип перечисления.Содержит определенный набор значений, перечисленных с именами.



void.
Тип void представляет собой отсутствие значения. В основном он используется для ф-ций, которые не возвращают значение.

    void func() {
    //code
    }


Указатель/ Pointer.
Указатель содержит адрес, который ссылается на объект заданного типа.

    int x {42};
    int* px = &x;
    cout << format("x is {}\n, *px);

В С++ указатели строго типизированы.
Тип указателя используется в качестве типа значения, на которое указывает ссылка.
Он также используется для определения размера приращений, убавлений и арифметических операций над указателем. 


Ссылка/ Reference.

    int x {42};
    int& rx = x;
    cout << format("x is {}\n, rx);

Ссылка похожа на указатель, но с другой семантикой. Ссылки неизменяемы. После определения ссылка не может быть изменена для ссылки на другой объект.
Доступ к ссылке осуществляется как к псевдонимам без какого-либо синтаксического указания на то, что это ссылка. 


auto.
Тип auto использует вывод типа для определения типа переменной.
    int y {42};
    auto x = y * 2;




Integers types/ Целочисленные типы.
-это простые примитивные (базовые) типы данных для представления целочисленных значений. Целые числа без дробной части.
Это:
1. char (1 байт = 8 бит)
2. short int (2 байта = 16 бит)
3. int (4 байта = 32 бит) 
4. long int (32 или 64 бит)
5. long long int (32 или 64 бита)
